## 0.0.1

本日志按“当前插件功能与行为”整理，聚焦 Matrix 平台的真实能力与配置结构。

### 核心能力
- 仅支持 Matrix 房间：群聊日常分析、图片/文本/PDF 报告输出。
- 自动分析调度：支持按时触发与并发控制，支持排除指定 bot 的 matrix 账号。
- 报告模板：支持多模板切换与预览，包含图片与 PDF 版本模板。
- LLM 分析：话题/用户称号/金句分析，支持独立 Provider 与提示词配置。

### 输出与发送
- 图片与 PDF 均使用 Matrix 上传接口发送。
- 图片发送失败支持重试队列；重试失败回退文本报告。
- PDF 输出固定保存在插件数据目录的 reports 下，不支持自定义输出目录。

### 配置结构
- 配置按用途分组：`group_access` / `auto_analysis` / `analysis` / `llm` / `output`。
- 分析子模块独立分组（话题/称号/金句），各自的提示词与 Provider 可配置。

### Matrix-only 行为
- 不再包含 QQ/OneBot 相关逻辑或参数（如群名片、合并转发、retcode 分支等）。
- 用户标识以 Matrix 用户 ID 形式保存与展示（字符串）。
